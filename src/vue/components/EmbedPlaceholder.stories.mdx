# Embed Placeholder

## Generic

This component is used as a generic social media embed placeholder. Furthermore it's the base component for our vendor specific placeholder components.

  components: { EmbedPlaceholder },
  template: `<EmbedPlaceholder :privacyManagerId="12345" />`,
}}></Story>
